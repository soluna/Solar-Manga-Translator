# Windows 运行与使用指南

恭喜您成功在 Windows 上启动了 Manga Auto-Translator 的 Web 界面！

## 1. 如何下载字体和模型？

虽然系统已经启动，但在第一次进行翻译时，后台需要调用字体和模型文件。请确保您已经完成了以下两步：

### 下载中文字体 (用于自动排版)
在项目根目录下（也就是 `main.py` 所在的文件夹），打开一个新的终端或命令提示符（确保进入了您刚刚创建的 `venv` 环境），运行：
```cmd
python download_models.py
```
这会自动帮您下载一款适合漫画排版的开源粗体中文字体（NotoSansSC-Bold）并放入 `fonts/` 文件夹。
*(如果您想用自己喜欢的字体，比如方正粗圆，也可以直接把它的 `.ttf` 或 `.otf` 文件扔进 `fonts/` 文件夹，并修改 `modules/typesetter.py` 里的 `font_path` 指向它。)*

### 下载漫画气泡检测模型
这个文件太大了，必须手动下载：
1. 点击打开这个链接: [comic-text-detector.pt (GitHub Releases)](https://github.com/zyddnys/manga-image-translator/releases/download/beta-0.2.1/comic-text-detector.pt)
2. 把下载下来的文件原封不动地放进项目的 **`models/`** 文件夹里。
*(如果您没有看到 `models` 文件夹，请自己新建一个)*

---

## 2. 要翻译的漫画放在哪里？

当前第一版的 Web 界面设计为了**让您快速测试单张漫画的汉化效果和调参**，因此不需要把漫画放到特定的文件夹！

### 如何使用：
1. 打开浏览器，访问控制台里打印出来的地址：`http://127.0.0.1:7860`
2. 在左侧的 **“LLM API Key”** 框中，填入您的 OpenAI（或其他兼容平台的）API Key。
3. 在下方的 **“上传漫画页”** 框中，直接把您想测试的一张日本漫画图片**拖拽进去**，或者点击它从电脑里选择一张图片。
4. 点击蓝色的 **“🚀 提取并翻译”** 按钮。

然后您就可以盯着右侧的“处理日志 (Console Log)”了。
如果是**第一次点击翻译**，程序会在后台自动联网从 HuggingFace 下载识别日文的最强视觉模型 `manga-ocr` (约 300MB)，所以请耐心等待一会儿。

下载完毕后，您就会在日志里看到整个流水线被激活：
1. 检测气泡框...
2. 识别日文...
3. 丢给大模型翻译...
4. 擦除原图文字...
5. 中文自动居中排版...

最后，一张完美的汉化漫画就会显示在右侧的预览框里了！快去试试吧！
